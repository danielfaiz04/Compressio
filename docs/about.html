About Compressio

A modern web application for file compression

What is Compressio?

Compressio is a modern web-based file compression application that allows users to compress and decompress various file types with a clean, user-friendly interface. The application features Google Authentication (Firebase) for secure access, session-based compression limits, and advanced options like AI-based method selection and sensitive content detection.

Features

  • File Compression

    Compress various file types including images, documents, videos, audio, and archives using modern algorithms.

  • Google Sign-In

    Secure authentication using Firebase and Google Sign-In.

  • Compression History

    View your recent compression activities, see compression ratio, and download or compare results.

  • Multiple Compression Methods

    Choose between AI Selection, Gzip, Brotli, WebP, PDF Optimize, Office Optimize, and Video Optimize. Each method is tailored for specific file types and use cases.

  • Compression Profiles

    Select from Default, Web, Archive, or Network profiles to optimize for your needs.

  • Sensitive Mode

    Enable sensitive mode to detect sensitive entities in text-based files using advanced NLP.

Supported File Types

Images

  • PNG (.png)
  • JPEG (.jpg, .jpeg)
  • WebP (.webp)

Audio

  • MP3 (.mp3)
  • WAV (.wav)

Documents

  • Text Files (.txt)
  • PDF Documents (.pdf)
  • Word (.doc, .docx)
  • Excel (.xls, .xlsx)
  • PowerPoint (.ppt, .pptx)

Video & Archives

  • MP4 (.mp4), MPEG (.mpeg), MOV (.mov)
  • ZIP (.zip), RAR (.rar)
  • JSON (.json), BIN (.bin)

Technology Used

Frontend

  • HTML5, CSS3, JavaScript (Vanilla)
  • Bootstrap 5
  • Firebase Authentication

Backend

  • Python 3
  • FastAPI, Uvicorn
  • SQLAlchemy (SQLite)
  • Pillow, OpenCV, scikit-image, pypdf, moviepy, brotli, gzip, python-magic, diff-match-patch, lxml, requests

How to Use Compressio

  1. Upload Your File: Drag and drop your file into the designated area or click "Choose Files" to select it from your device.
  2. Select Compression Method & Profile: Choose your preferred compression method (AI, Gzip, Brotli, WebP, PDF, Office, Video) and profile (Default, Web, Archive, Network).
  3. (Optional) Enable Sensitive Mode: Detect sensitive content in text-based files.
  4. Process File: Click the "Process File" button to start the compression.
  5. Download or Compare Result: Once processed, the compressed file will appear in the "Compression History" table. Click "Download" or "Compare" to view details.
