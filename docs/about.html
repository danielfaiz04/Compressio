About Compressio

A modern web application for file compression

What is Compressio?

Compressio is a web-based file compression application that allows users to compress and decompress files with a clean, user-friendly interface. The application features Firebase Authentication for secure access and session-based compression limits.

Features

  • File Compression

    Compress various file types including images and documents using different algorithms.

  • Google Sign-In

    Secure authentication using Firebase and Google Sign-In.

  • Compression History

    View your recent compression activities and download results.

  • Multiple Algorithms

    Choose between Huffman coding and Run-Length Encoding based on your needs.

Supported File Types

Images

  • PNG (.png)
  • JPEG (.jpg, .jpeg)

Documents

  • Text Files (.txt)
  • PDF Documents (.pdf)

Technology Used

Frontend

  • HTML5
  • CSS3
  • JavaScript (Vanilla)
  • Bootstrap 5
  • Firebase Authentication

Backend

  • Python 3
  • Flask
  • Flask-CORS
  • Pillow (for image processing)

How to Use Compressio

  1. Upload Your File: Drag and drop your file into the designated area or click "Choose Files" to select it from your device.
  2. Select Compression Type: Choose between "Compress" or "Decompress" based on your needs.
  3. Choose Algorithm (Optional): Select your preferred compression algorithm. Auto (automatically selects the best algorithm for your file), Huffman Coding (an efficient method for lossless data compression), or Run-Length Encoding (a form of data compression in which runs of data are stored as a single data value and count).
  4. Process File: Click the "Process File" button to start the compression or decompression.
  5. Download Result: Once processed, the compressed/decompressed file will appear in the "Recent Compressions" table. Click "Download" to save it.
